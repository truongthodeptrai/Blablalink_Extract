Same as Nikke_stats_scraper, here are the requirements to run scraper.js in Nikke_UR_scraper

- Type `npm install` in Terminal

Here's how to run scraper.js in Nikke_UR_scraper

- Type `node scraper.js --login`
- Login your blablalink account
- Exit the website so that the cookies will be saved in the project
- Type `node scraper.js`
- Wait for a few minutes to get the result
- Open the html file then choose the csv file, and it's done
